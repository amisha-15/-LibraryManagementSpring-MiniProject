# Library Management System (Spring Core)

## Features
- Add Book
- Issue Book
- Return Book
- View Books

## Technologies
- Java
- Spring Core
- Maven

## Concepts Used
- IoC (Inversion of Control)
- Dependency Injection (DI)
- @Service, @Repository
- Java Configuration
- Exception Handling
- Layered Architecture

- ## Project Flow
User → Service Layer → Repository → Data Storage

## How to Run
1. Clone the repo
2. Open in IntelliJ
3. Run App.java

## Key Highlights
- Constructor-based Dependency Injection
- Layered Architecture (Service, Repository)
- Exception Handling implemented
- Input validation added
